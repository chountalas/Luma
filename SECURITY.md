# Security Policy

## Reporting A Vulnerability

Please report suspected security issues privately by opening a GitHub security advisory for this repository or by contacting the maintainer directly through the GitHub profile linked to the repository owner.

Do not open public issues for vulnerabilities that could put users at risk. Include the affected version or commit, reproduction steps, expected impact, and any relevant logs or screenshots.

## Scope

Luma is a local macOS menu bar app. Security reports are most useful when they involve unsafe local file access, persistence behavior, signing or release packaging issues, or behavior that could unexpectedly expose user data.
